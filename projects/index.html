Flow-Gated Structure from Motion with Dense UFM Correspondences

Overview

Flow-SfM is a flow-gated Structure-from-Motion framework that combines dense UFM correspondences with optical flow to robustly estimate camera poses and reconstruct dense 3D scene geometry in challenging multi-view settings.

By leveraging dense correspondences rather than only sparse keypoints, Flow-SfM aims to improve robustness in low-texture or repetitive regions, while the flow-gating mechanism filters unreliable matches using temporal motion cues.

We propose a dense correspondence based Structure-from-Motion pipeline that integrates optical flow to gate unreliable matches. Dense UFM correspondences are first extracted across views, followed by flow-guided filtering that suppresses inconsistent matches. The remaining correspondences are used for robust camera pose estimation and dense 3D structure recovery, leading to improved performance on challenging scenes.

Method

The Flow-SfM pipeline consists of:

  • Dense feature extraction and UFM correspondences across multiple views
  • Optical flow estimation between consecutive frames
  • Flow-gated filtering of inconsistent correspondences
  • Robust pose estimation and triangulation using the filtered matches
  • Refinement and dense reconstruction of the scene structure
